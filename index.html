  • Alt + 4 - Light Theme
  • Alt + R - Reset Theme

About This Page

This is a demo webpage designed to test accessibility features for visually impaired users. You can change the color theme using the buttons above or keyboard shortcuts.

The page demonstrates how different color schemes can improve readability and reduce eye strain for users with different visual needs.
